# ToolBox v1.8.8

## 修复

- 修复 Android 14 上局域网文件互传前台服务启动时缺少服务类型导致崩溃的问题。
- 修复 Android 10+ 文件互传列表可见但下载 404 的问题，改为通过 MediaStore content Uri 读取文件。
- 修复 Android 13+ 视频下载被不必要的媒体读取权限拦截的问题。
- 修复金属探测器弱震动报警参数长度不匹配导致的崩溃。
- 修复视频下载页面退出后后台进度回调仍更新 UI 的生命周期风险。
- 修复视频转 GIF 输出 Uri 查询不可靠、失败时残留 pending 媒体项的问题。
- 修复 Android 8/9 上二维码、九宫格、GIF、视频转 GIF、文件互传保存前缺少写存储权限的问题。
- 修复 WiFi 热力图申请步频权限后重复启动采样导致传感器重复注册的问题。

## 清理

- 清理构建 warning 和局部代码卫生问题，保持 debug/release 构建日志干净。
